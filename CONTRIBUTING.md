# Kontribusi ke LaraCarte

Terima kasih telah tertarik untuk berkontribusi pada LaraCarte! Kami menyambut segala bentuk kontribusi, mulai dari laporan bug, saran fitur, hingga perbaikan kode.

## Cara Berkontribusi

### 1. Pelaporan Masalah (Issues)
Jika Anda menemukan bug atau memiliki ide fitur, silakan buka Issue di repositori ini. Pastikan untuk:
*   Memberikan deskripsi yang jelas.
*   Menyertakan langkah-langkah untuk mereproduksi bug (jika ada).
*   Menyebutkan versi sistem yang digunakan.

### 2. Pull Requests (PR)
Kami sangat menghargai kontribusi kode. Ikuti langkah berikut untuk mengirimkan PR:
1.  **Fork** repositori ini.
2.  Buat **Branch Baru** untuk fitur atau perbaikan Anda (`git checkout -b fitur/nama-fitur`).
3.  Lakukan **Commit** perubahan Anda dengan pesan yang deskriptif.
4.  **Push** ke branch Anda di GitHub.
5.  Buka **Pull Request** ke branch `main` repositori asli.

## Standar Kode

Untuk menjaga kualitas kode tetap terjaga, kami harap Anda mengikuti panduan berikut:
*   Gunakan standar penamaan Laravel (camelCase untuk method, PascalCase untuk class).
*   Gunakan indentasi 4 spasi (default Laravel).
*   Sertakan komentar pada logika yang kompleks.
*   Pastikan kode Anda lulus tes fitur yang ada (`php artisan test`).

## Komunikasi

Jika Anda memiliki pertanyaan lebih lanjut, silakan hubungi tim pengelola melalui tab Discuss atau Issues.
